# Markdown Parser

## Description
Implement the core parsing logic to read story files, extract YAML frontmatter, and separate the markdown content.

## Acceptance Criteria
- [ ] `Parser.parse(content)` returns `Story` object with correct metadata and content
- [ ] Throws specific error for invalid YAML frontmatter
- [ ] Throws specific error for missing required fields (id, title, type)
- [ ] Correctly parses dates from strings to Date objects
- [ ] Extracted markdown content excludes frontmatter
- [ ] Can parse Epic files into `Epic` objects
