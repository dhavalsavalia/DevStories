# Auto-Timestamps on Save

## Description
Automatically update the `updated` field in the frontmatter whenever a story file is saved.

## Acceptance Criteria
- [x] Saving a story file triggers the `onWillSaveTextDocument` or `onDidSaveTextDocument` handler
- [x] The `updated` field in frontmatter is changed to current date (YYYY-MM-DD)
- [x] If `updated` field is missing, it is added
- [x] Other frontmatter fields and markdown content remain unchanged
- [x] Does not trigger infinite loop with File Watcher
