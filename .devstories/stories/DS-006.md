# TreeDataProvider

## Description
Implement the VS Code TreeDataProvider interface to render the stories in the sidebar.

## Acceptance Criteria
- [x] `TreeDataProvider.getChildren()` returns Epics when element is undefined
- [x] `TreeDataProvider.getChildren(epic)` returns Stories belonging to that Epic
- [x] `TreeDataProvider.getTreeItem(element)` returns correct UI properties
- [x] Tree view refreshes automatically when Store emits update event
- [x] Empty state (no stories) is handled gracefully
