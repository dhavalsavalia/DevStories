# Progress Bar (Status Bar)

## Description
Display a progress bar in the VS Code status bar showing the completion status of the current sprint.

## Acceptance Criteria
- [x] Status bar item appears on extension activation
- [x] Displays "Sprint X: [Progress Bar] Y/Z"
- [x] Calculates total stories in current sprint
- [x] Calculates completed stories (status=done)
- [x] Updates immediately when a story status changes
- [x] Hides if no active sprint is configured
