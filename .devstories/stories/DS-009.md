# Progress Bar (Status Bar)

## Description
Display a progress bar in the VS Code status bar showing the completion status of the current sprint.

## Acceptance Criteria
- [ ] Status bar item appears on extension activation
- [ ] Displays "Sprint X: [Progress Bar] Y/Z"
- [ ] Calculates total stories in current sprint
- [ ] Calculates completed stories (status=done)
- [ ] Updates immediately when a story status changes
- [ ] Hides if no active sprint is configured
