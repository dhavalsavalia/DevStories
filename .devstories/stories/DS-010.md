# Init Command

## Description
Command to initialize the `.devstories` directory structure in a new project.

## Acceptance Criteria
- [ ] `init` command creates `.devstories` directory if missing
- [ ] Creates `config.yaml` with default content
- [ ] Creates `epics` and `stories` subdirectories
- [ ] Prompts user to add `.devstories` to `.gitignore` (optional but recommended)
- [ ] Does not overwrite existing configuration without confirmation
