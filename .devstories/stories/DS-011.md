# Create Epic Command + Auto-Link

## Description
Command to create a new epic with smart defaults and automatic story linking support.

## Acceptance Criteria

### MVP
- [ ] `devstories.createEpic` command registered
- [ ] Prompts for Epic title
- [ ] Generates unique Epic ID based on config prefix
- [ ] Creates markdown file in `epics/` folder with template
- [ ] Opens the newly created file in editor
- [ ] Tree view refreshes to show new epic

### Enhanced
- [ ] **Goal prompt**: After title, ask "What's the main goal?" - pre-fills description
- [ ] **Sprint picker**: Quick pick for existing sprints or "Create new sprint"
- [ ] **Stories section**: Include placeholder `## Stories` section for auto-linking
- [ ] **Duplicate check**: Warn if epic with similar title exists

## Technical Notes
- ID generation: Find highest existing ID, increment by 1
- Use `vscode.workspace.openTextDocument()` then `showTextDocument()`
- Epic template should match PRD spec format

## Epic Template

```markdown
---
id: {{ID}}
title: "{{TITLE}}"
status: todo
sprint: {{SPRINT}}
created: {{DATE}}
---

# {{TITLE}}

## Description
{{GOAL_OR_PLACEHOLDER}}

## Acceptance Criteria
- [ ]

## Stories
<!-- Stories will be auto-linked here when created -->

## Notes

```
