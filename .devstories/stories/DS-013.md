# Quick Capture

## Description
A keyboard shortcut (Cmd+Shift+S) to quickly create a story without losing context.

## Acceptance Criteria
- [ ] Global keybinding `Cmd+Shift+S` triggers Quick Capture
- [ ] Shows simple input box for title
- [ ] Creates story with default settings (Type=Task, Size=M, current Sprint)
- [ ] Does not interrupt current editing flow (doesn't open file, just notifies)
- [ ] Tree view updates immediately to show new story
