# Kanban Board UI

## Description
Beautiful, functional Kanban board with columns, cards, and essential information at a glance. Designed for developer aesthetics.

## Acceptance Criteria

### MVP
- [ ] Board renders columns from config.yaml statuses
- [ ] Stories render as cards in correct columns based on status
- [ ] Cards display: ID, Title, Type icon, Size badge
- [ ] Columns show count of stories
- [ ] Horizontal scroll for many columns, vertical scroll within columns
- [ ] Empty column state: "No stories" placeholder

### Enhanced
- [ ] **WIP limits**: Configurable max items per column, visual warning when exceeded
- [ ] **Swimlanes by Epic**: Toggle to group stories by epic (horizontal lanes)
- [ ] **Compact/expanded view**: Toggle card detail level
- [ ] **Story count by type**: Column header shows "3 features, 1 bug"
- [ ] **Color coding**: Subtle color hints based on type or priority
- [ ] **Card hover**: Show full title and dependencies on hover

## Technical Notes
- Use CSS Grid for column layout
- Cards: Flexbox for internal layout
- Scrolling: `overflow-x: auto` for board, `overflow-y: auto` for columns
- Icons: Inline SVG or VS Code codicons
