# Load Custom Templates into Picker

## Description
Custom templates saved to `.devstories/templates/` should appear in the template picker when creating stories.

## Acceptance Criteria

### MVP
- [x] Scan `.devstories/templates/` folder when createStory runs
- [x] Show custom templates in picker alongside bundled ones
- [x] Custom templates grouped under "Custom" section in picker
- [x] Template filename (minus .md) used as display name

### Enhanced
- [x] Template metadata from YAML frontmatter (optional title, description)
- [x] Filter templates by story type (if specified in frontmatter)

## Implementation Notes
- Added `CustomTemplate` interface and `parseCustomTemplate()` function to `createStoryUtils.ts`
- Added `loadCustomTemplates()` to scan `.devstories/templates/` folder
- Template picker shows custom templates under separator when relevant to selected story type
- Templates without `types` frontmatter appear for all story types
- 6 new unit tests + 2 integration tests added
