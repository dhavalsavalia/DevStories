# Load Custom Templates into Picker

## Description
Custom templates saved to `.devstories/templates/` should appear in the template picker when creating stories.

## Acceptance Criteria

### MVP
- [ ] Scan `.devstories/templates/` folder when createStory runs
- [ ] Show custom templates in picker alongside bundled ones
- [ ] Custom templates grouped under "Custom" section in picker
- [ ] Template filename (minus .md) used as display name

### Enhanced
- [ ] Template metadata from YAML frontmatter (optional title, description)
- [ ] Filter templates by story type (if specified in frontmatter)
