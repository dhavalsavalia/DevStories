# Wrap dependencies in [[ID]] link format

## Bug Description
When creating a story with dependencies, the frontmatter stores them as plain IDs (e.g., `STORY-006`) instead of link format (e.g., `[[STORY-006]]`). This is inconsistent with the link system used elsewhere.

## Expected vs Actual
- **Expected:** `dependencies: - [[STORY-006]]`
- **Actual:** `dependencies: - STORY-006`

## Root Cause
`generateStoryMarkdown()` in createStoryUtils.ts joins dependencies without wrapping in `[[]]`.

## Fix
Updated dependency formatting in `generateStoryMarkdown()` to wrap each ID in `[[]]`:
```typescript
data.dependencies.map(d => `[[${d}]]`).join('\n  - ')
```
