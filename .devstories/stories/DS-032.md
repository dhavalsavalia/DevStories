# Sprint management panel

## Description

A webview panel to view, edit, and manage all sprints. Shows sprint list with actions (start, close, edit, delete) and completed sprint summaries.

## User Flow

1. **Trigger**: Command "DevStories: Manage Sprints" or status bar â†’ Manage
2. **Panel Opens**: Shows all sprints grouped by status
3. **Actions** per sprint:
   - **Planned**: Start, Edit, Plan, Delete
   - **Active**: Close, Edit
   - **Completed**: View summary (read-only)

### Sprint Cards

Each sprint shows:
- Name and dates
- Status badge (active/planned/completed)
- Goal (if set)
- Progress (active) or story count (planned) or summary (completed)
- Action buttons based on status

| Active    | Complete, Edit             |
| Planned   | Start, Plan, Edit, Delete  |
