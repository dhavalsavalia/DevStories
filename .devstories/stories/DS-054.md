# Identify and remove unused exports/dependencies

## Description
Run static analysis tools to find and remove dead code, unused exports, and stale dependencies.

## Acceptance Criteria

- [ ] Run `npx depcheck` to find unused dependencies
- [ ] Remove identified dead code
- [ ] Update imports after removals
- [ ] Document any intentionally unused re-exports (if any)
- [ ] All tests still pass after cleanup
