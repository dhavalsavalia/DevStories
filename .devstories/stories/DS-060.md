# Implement accessibility labels in board webview

## Description
Add ARIA labels to all interactive elements in the board webview for screen reader support.

## Acceptance Criteria
- [ ] Add `aria-label` to story cards with full context (title, status, size)
- [ ] Add `role="region"` to columns with story count
- [ ] Add `aria-label` to search input
- [ ] Add `aria-label` to all filter dropdowns
- [ ] Add visible keyboard focus indicators
- [ ] Test with VoiceOver/NVDA screen reader
