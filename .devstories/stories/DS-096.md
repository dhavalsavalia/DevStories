# Normalize dependency format across all stories

## Description

Convert all story dependency fields to use the canonical format: block array with wiki-links.

## Acceptance Criteria

- [ ] All dependencies use block array format (not inline)
- [ ] All story IDs wrapped in `[[ID]]` syntax
- [ ] Empty dependencies use `dependencies: []`
- [ ] No orphaned or broken dependency links

## Technical Notes

Canonical format:
```yaml
dependencies:
  - [[DS-001]]
  - [[DS-002]]
```

Not:
```yaml
dependencies: [DS-001, DS-002]
```
