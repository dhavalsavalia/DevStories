# Autocomplete for enum fields

## Description

Provide autocomplete suggestions for frontmatter fields with known values.

## Acceptance Criteria

- [ ] `status:` suggests values from config.yaml statuses
- [ ] `type:` suggests feature, bug, task, chore
- [ ] `size:` suggests values from config.yaml sizes
- [ ] Suggestions appear when typing in frontmatter YAML
- [ ] Suggestions show description/label where available

## Technical Notes

- CompletionItemProvider for markdown files in .devstories/
- Detect cursor position in frontmatter (between `---` markers)
- Parse field name to determine which suggestions to show
