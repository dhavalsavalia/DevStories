# Autocomplete for enum fields

## Description

Provide autocomplete suggestions for frontmatter fields with known values.

## Acceptance Criteria

- [ ] `status:` suggests values from config statuses
- [ ] `type:` suggests feature, bug, task, chore
- [ ] `size:` suggests values from config sizes
- [ ] `sprint:` suggests values from config sprints.sequence
- [ ] Suggestions appear when typing in frontmatter YAML
- [ ] Suggestions show description/label where available

## Technical Notes

- Create `src/providers/frontmatterCompletionProvider.ts`
- Register with trigger chars `:` and ` ` (space)
- Reuse `isInFrontmatter()` from `storyHoverProviderUtils.ts`
- Read ConfigService for dynamic values
