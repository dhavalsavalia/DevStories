# Autocomplete for references

## Description

Provide autocomplete for fields and syntax that reference other stories/epics.

## Acceptance Criteria

- [x] `epic:` field suggests existing epic IDs
- [x] `dependencies:` array suggests existing story IDs
- [x] `[[` anywhere in file triggers story/epic ID suggestions
- [x] Suggestions show title alongside ID
- [x] Suggestions filter as user types
