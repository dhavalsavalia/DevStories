# Autocomplete for references

## Description

Provide autocomplete for fields and syntax that reference other stories/epics.

## Acceptance Criteria

- [ ] `epic:` field suggests existing epic IDs
- [ ] `dependencies:` array suggests existing story IDs
- [ ] `[[` anywhere in file triggers story/epic ID suggestions
- [ ] Suggestions show title alongside ID
- [ ] Suggestions filter as user types

## Technical Notes

- Extend CompletionItemProvider from DS-123
- Query Store for available IDs
- Handle `[[` trigger character for link completion
