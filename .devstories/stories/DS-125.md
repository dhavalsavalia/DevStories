# Hover documentation for frontmatter

## Description

Show helpful information when hovering over frontmatter fields and references.

## Acceptance Criteria

- [ ] Hover on `[[EPIC-ID]]` shows: title, status, story count
- [ ] Hover on `epic:` value shows epic details
- [ ] Hover content is formatted markdown
