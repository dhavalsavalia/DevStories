# Create issue templates

## Description

Add form-based issue templates for bug reports and feature requests.

## Acceptance Criteria

- [x] Create `.github/ISSUE_TEMPLATE/bug_report.yml` (form-based)
  - Required: VS Code version, extension version, OS
  - Required: Steps to reproduce
  - Optional: Expected vs actual behavior
  - Optional: Screenshots
- [x] Create `.github/ISSUE_TEMPLATE/feature_request.yml`
  - Required: Problem description
  - Optional: Proposed solution
  - Optional: Alternatives considered
- [x] Create `.github/ISSUE_TEMPLATE/config.yml` to disable blank issues and link to discussions
