# Configure tag protection for releases

## Description

Protect version tags from being deleted or modified by non-maintainers.

## Acceptance Criteria

- [ ] Create tag ruleset in GitHub (Settings → Rules → Rulesets)
- [ ] Pattern: `v*` to protect all version tags
- [ ] Only maintainers/admins can create tags
- [ ] Prevent tag deletion
- [ ] Document in CONTRIBUTING.md
