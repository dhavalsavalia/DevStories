# Configure tag protection for releases

## Description

Protect version tags from being deleted or modified by non-maintainers.

## Acceptance Criteria

- [x] Create tag ruleset in GitHub (Settings → Rules → Rulesets)

- [x] Only maintainers/admins can create tags
- [x] Prevent tag deletion
- [x] Document in CONTRIBUTING.md
